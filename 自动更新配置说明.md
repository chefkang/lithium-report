# 锂盐报告自动更新 - Windows任务计划程序配置

## 方法：使用Windows任务计划程序（推荐）

### 步骤1：创建定时任务

打开 PowerShell（管理员），运行以下命令：

```powershell
# 创建每日09:00运行的任务
$action = New-ScheduledTaskAction -Execute "uv" -Argument "run python `"$env:USERPROFILE\.openclaw\workspace\update_lithium_report.py`"" -WorkingDirectory "$env:USERPROFILE\.openclaw\workspace"

$trigger = New-ScheduledTaskTrigger -Daily -At "09:00"

$settings = New-ScheduledTaskSettingsSet -AllowStartIfOnBatteries -DontStopIfGoingOnBatteries -StartWhenAvailable

Register-ScheduledTask -TaskName "锂盐报告自动更新" -Action $action -Trigger $trigger -Settings $settings -Description "每日09:00自动更新锂盐全球分析报告"
```

### 步骤2：验证任务创建

```powershell
# 查看已创建的任务
Get-ScheduledTask -TaskName "锂盐报告自动更新" | Get-ScheduledTaskInfo
```

### 步骤3：手动运行测试

```powershell
# 立即运行一次测试
Start-ScheduledTask -TaskName "锂盐报告自动更新"

# 查看运行结果
Get-WinEvent -FilterHashtable @{LogName='Microsoft-Windows-TaskScheduler/Operational'; ID=201} | Select-Object -First 5
```

### 步骤4：推送到GitHub Pages

为了让网站更新，需要将生成的 `index.html` 推送到你的 GitHub 仓库。

在你的 GitHub 仓库根目录创建 `.github/workflows/update-report.yml`：

```yaml
name: Auto Update Lithium Report

on:
  schedule:
    - cron: '0 1 * * *'  # 每天UTC 01:00 (北京时间09:00)
  workflow_dispatch:

jobs:
  update:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      
      - name: Setup Python
        uses: actions/setup-python@v4
        with:
          python-version: '3.11'
      
      - name: Update Report
        run: python update_lithium_report.py
      
      - name: Commit and Push
        run: |
          git config user.name "GitHub Actions"
          git config user.email "actions@github.com"
          git add index.html update_log.txt
          git commit -m "Auto update lithium report - $(date +%Y-%m-%d)" || exit 0
          git push
```

### 替代方案：使用OpenClaw Heartbeat

如果你希望使用OpenClaw内置的定时功能，编辑 `HEARTBEAT.md`：

```markdown
# Heartbeat Tasks

## 每日09:00更新锂盐报告
- 运行: python update_lithium_report.py
- 推送: git push origin main
- 记录: memory/lithium-updates.md
```

然后OpenClaw会在每天09:00自动唤醒执行。

## 验证自动更新

1. **立即测试**: 双击运行 `update_lithium_report.py`
2. **检查输出**: 查看是否生成了新的 `index.html`
3. **查看日志**: 检查 `update_log.txt` 中的更新记录
