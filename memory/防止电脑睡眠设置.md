# 防止电脑睡眠 - 保持AI后台运行

## 方法一：快速设置（推荐）

以管理员身份运行 PowerShell，执行：

```powershell
# 关闭睡眠模式
powercfg /change standby-timeout-ac 0
powercfg /change standby-timeout-dc 0

# 关闭显示器（可选，保持0为从不关闭）
powercfg /change monitor-timeout-ac 0

# 禁止硬盘休眠
powercfg /change disk-timeout-ac 0

# 查看当前设置
powercfg /query SCHEME_CURRENT SUB_SLEEP
```

## 方法二：GUI设置

1. 右键开始菜单 → **设置**
2. **系统** → **电源和睡眠**
3. 设置：
   - **屏幕**：`从不` 关闭
   - **睡眠**：`从不`
4. **其他电源设置** → **更改计划设置** → **更改高级电源设置**
   - 硬盘 → 关闭硬盘 → 设置为 `0`（从不）
   - 睡眠 → 允许混合睡眠 → `关闭`

## 方法三：使用Insomnia（推荐长时间运行）

下载 [Insomnia](https://dl.hexchat.net/insomnia/) 或使用命令：

```powershell
# 临时阻止睡眠（运行后保持窗口打开）
Add-Type -TypeDefinition @"
using System;
using System.Runtime.InteropServices;
public class Power {
    [DllImport("kernel32.dll")]
    public static extern uint SetThreadExecutionState(uint esFlags);
    public const uint ES_CONTINUOUS = 0x80000000;
    public const uint ES_SYSTEM_REQUIRED = 0x00000001;
    public const uint ES_DISPLAY_REQUIRED = 0x00000002;
}
"@
[Power]::SetThreadExecutionState([Power]::ES_CONTINUOUS -bor [Power]::ES_SYSTEM_REQUIRED -bor [Power]::ES_DISPLAY_REQUIRED)
Write-Host "已阻止电脑睡眠，按 Ctrl+C 取消"
while($true) { Start-Sleep -Seconds 60 }
```

## 验证设置
```powershell
powercfg /requests
```
如果有活动请求，电脑就不会睡眠。
