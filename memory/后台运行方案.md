# 保持 AI 后台运行方案

## 方案一：防止电脑睡眠（推荐短期使用）

### Windows 设置修改：
1. **设置 → 系统 → 电源和睡眠**
2. **屏幕：** 设置为 "从不" 关闭
3. **睡眠：** 设置为 "从不"
4. **控制面板 → 电源选项 → 更改计划设置 → 更改高级电源设置**
   - 硬盘 → 在此时间后关闭硬盘 → 设置为 0（从不关闭）
   - 睡眠 → 允许混合睡眠 → 关闭

### 或者使用命令行临时阻止睡眠：
```powershell
# 阻止电脑睡眠（需要在后台运行）
powercfg /requestsoverride PROCESS claw.exe DISPLAY SYSTEM
```

## 方案二：使用 OpenClaw 的 Heartbeat / Cron 机制

### Heartbeat（适合频繁检查）
编辑 `HEARTBEAT.md`，添加周期性任务，OpenClaw 会定期唤醒我检查状态。

### Cron（适合定时任务）
设置定时任务，让 AI 在特定时间自动运行，不需要保持电脑一直开机。

## 方案三：使用服务方式运行（推荐长期使用）

将 OpenClaw 设置为 Windows 服务，即使登录注销也能运行。
