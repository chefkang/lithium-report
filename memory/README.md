# 小康的记忆系统 v1.0

## 每日工作流程

### 每天结束时（或重要节点），AI 必须：
1. **创建当日记忆文件** `memory/YYYY-MM-DD.md`
2. **记录内容：**
   - 今天完成了什么
   - 进行中的任务和进度
   - 明天需要继续做的事
   - 重要的决策和发现
   - 遇到的问题和解决方案

### 每天开始时，AI 必须：
1. 读取 `memory/昨天日期.md` 了解昨天的工作
2. 读取 `memory/今天日期.md`（如果有）
3. 读取 `MEMORY.md` 长期记忆
